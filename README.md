# Flames Calculator

🚀 A simple and practical **Spring Boot** web application built with **Java**, **Thymeleaf**, and **MySQL**, and deployed on **Railway** (both backend and database hosted).

## 🔗 Live App
👉 https://spring-boot-basics-production-765c.up.railway.app/

> This is the deployed version of the project hosted on Railway.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Java (JDK 17+) |
| Framework | Spring Boot |
| Database | MySQL |
| UI | Thymeleaf |
| Build Tool | Maven |
| Hosting | Railway.app (backend + MySQL DB) |

---

## 📂 Project Overview

This repository contains a Spring Boot based web application that:

✅ Uses **Spring MVC** with **Thymeleaf templates** for server-side HTML rendering  
✅ Connects to a **MySQL** database for data persistence  
✅ Is packaged and deployed to **Railway**, including a hosted MySQL database  
✅ Demonstrates CRUD and standard web functionality (your app logic)

---

## 🧩 Features (Example)

✔ Home page and navigation  
✔ Database backed user/data management  
✔ Dynamic HTML using Thymeleaf  
✔ Standard Spring Boot controllers and services  
✔ Configuration via `application.properties`  
✔ Railway environment support via environment variables
